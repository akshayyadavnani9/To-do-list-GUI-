# 📌 To-Do List GUI Application

## 📖 Description
This is a simple To-Do List Desktop Application built using Python and Tkinter.  
It allows users to add and delete tasks easily through a graphical user interface (GUI).

This project is created as part of the Codetech Internship Program.

---

## 🚀 Features
- Add new tasks  
- Delete selected tasks  
- Simple and user-friendly GUI  
- Lightweight and fast  

---

## 🛠️ Technologies Used
- Python  
- Tkinter (GUI Library)  

---

## 📂 Project Structure
```
To-Do-List-GUI/
│── todo.py
│── README.md
```

---

## ▶️ How to Run the Project

1. Install Python (if not installed)  

2. Clone this repository:
```
git clone https://github.com/akshayyadavani9/To-do-list-GUI.git
```

3. Navigate to the folder:
```
cd To-do-list-GUI
```

4. Run the program:
```
python todo.py
```

---

## 📸 Output
A window will open where you can:
- Enter tasks  
- Add tasks  
- Delete selected tasks
